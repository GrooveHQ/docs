# conversationStar

Star a Conversation.

For example:

```
mutation(
  $changesetId: String,
  $conversationId: ID!
) {
  conversationStar(
  input: {
    changesetId: $changesetId,
    conversationId: $conversationId
  }
  ) {
  changesetId
  conversation {
    ... on Conversation {
      id
      starred
    }
  }
  diffs {
    query
    delta
  }
  errors {
    path
    message
  }
  }
}
```

To un-star a conversation, see `conversationUnstar`
