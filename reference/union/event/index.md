# Event

A Groove Event represents a notable event triggered by some Actor.

Events generally apply to (or have a 'subject' of type):
  - Conversations
  - Messages
  - Ratings
  - Agents or Contacts

For example:
  - Message added
  - Note added
  - Conversation forwarded
  - Mailbox changed
  - Group changed
  - State changed
  - Starred
  - Unstarred
